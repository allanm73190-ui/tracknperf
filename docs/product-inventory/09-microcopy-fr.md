# 09 — Microcopy (FR) — Spec

> Objectif: textes cohérents, courts, “sport premium”, sans jargon.

## Global
- **Loading**: `Chargement…`
- **Retry**: `Réessayer`
- **Back**: `Retour`
- **Sign out**: `Se déconnecter`
- **Sync now**: `Synchroniser`
- **Synced**: `Synchronisé`
- **Queued**: `En attente`

## Auth (`/auth`)
- **Titre**: `Connexion`
- **Sign in**: `Se connecter`
- **Sign up**: `Créer un compte`
- **Magic link**: `Lien magique`
- **Email**: `Email`
- **Password**: `Mot de passe`
- **Continue**: `Continuer`
- **Send link**: `Envoyer le lien`
- **Success magic link**: `Lien envoyé. Vérifie ta boîte mail.`
- **Signup confirm**: `Vérifie ton email pour confirmer ton compte (si demandé).`
- **Supabase misconfig**: `Configuration Supabase manquante.`
  - détail: `Renseigne VITE_SUPABASE_URL et VITE_SUPABASE_ANON_KEY.`

## Callback (`/auth/callback`)
- **Completing**: `Finalisation de la connexion…`
- **No session**: `Aucune session trouvée. Réessaie de te connecter.`

## Onboarding (`/onboarding`)
- **Titre**: `Bienvenue`
- **Sous-titre**: `Créons ton profil (30 secondes).`
- **Email**: `Email`
- **Display name**: `Nom affiché`
- **Avatar URL**: `URL d’avatar (optionnel)`
- **CTA**: `Continuer`
- **Error no user**: `Tu dois être connecté pour continuer.`
- **Error misconfig**: `Supabase n’est pas configuré.`

## Today (`/today`)
- **Planned title**: `Plan du jour`
- **Empty planned**: `Aucune séance planifiée aujourd’hui.`
- **Recommended title**: `Recommandation`
- **No reco**: `Pas de recommandation pour le moment.`
- **Details**: `Détails`
- **Log title**: `Journal de séance`
- **Notes**: `Notes`
- **No planned link**: `Aucune séance planifiée liée.`

## History (`/history`)
- **Title**: `Historique`
- **Empty**: `Aucune séance enregistrée sur cette période.`
- **Open**: `Ouvrir`

## Session detail (`/session/:id`)
- **Title**: `Séance`
- **Not found**: `Séance introuvable (ou accès refusé).`
- **Missing id**: `Identifiant de séance manquant.`
- **What happened**: `Ce qui s’est passé`
- **Context**: `Contexte`
- **Payload**: `Données`
- **Back to history**: `Retour à l’historique`

## Stats (`/stats`)
- **Title**: `Stats`
- **Sessions**: `Séances`
- **Time**: `Temps`
- **Empty**: `Pas assez de données pour afficher des stats.`

## Admin (`/admin`)
- **Title**: `Admin`
- **Engine context**: `Contexte moteur`
- **Import a plan**: `Importer un plan`
- **Parse & preview**: `Analyser & prévisualiser`
- **Import**: `Importer`
- **Preview ready**: `Prévisualisation prête`
- **No preview**: `Aucune prévisualisation`
- **Create config profile**: `Créer un profil de config`
- **Create algorithm version**: `Créer une version d’algo`
- **Access denied**: `Accès refusé.`
