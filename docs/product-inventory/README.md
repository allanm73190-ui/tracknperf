# TrackNPerf — Product Inventory (UI/UX Design Pack)

Ce dossier regroupe **toutes les informations produit** utiles pour travailler dans un outil de design (Figma, Framer, etc.) sans ouvrir le code.

## Contenu
- `01-app-overview.md`: vision rapide, navigation, rôles, états globaux
- `02-pages-routes.md`: toutes les pages (routes) + contenu + états + erreurs
- `03-features.md`: features “métier” (import, reco, sync, etc.)
- `04-data-model.md`: entités/tables principales + relations utiles UI
- `05-auth-roles-permissions.md`: auth, onboarding, admin, permissions
- `06-sync-offline.md`: sync, outbox, idempotency (ce que l’UI doit montrer)
- `07-engine-recommendations.md`: recommandations + explainability (ce qui est affiché)

## Convention
- **Route**: URL dans l’app
- **UI sections**: blocs visibles principaux
- **States**: loading / empty / error / success
- **Actions**: boutons, formulaires, navigations
